How to install

  1. Install the latest MelonLoader version to DesktopMate (v0.6.6 is the recommended one)
  2. Install .NET Runtime 6.0 from Microsoft
  3. Download Custom Avatar Loader
  4. Extract the contents in the Desktop Mate folder

Usage

  1. Initial startup of Desktop Mate will take a while as IL2CPP needs to decompile the game binary
  2. After that, click on your character and press F4
  3. A file dialog will open. Choose any VRM file that you want (make sure it's in a folder that you have permission)
  4. And your model should immediately appear!
